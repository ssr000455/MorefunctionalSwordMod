# 更多扩展模组 (MoreFunctionalSwordMod)

一个为 Minecraft 1.20.1 设计的 Fabric 模组，添加了多种钻石工具、彩虹宝石系列装备、七彩神剑以及反作弊系统。

## 版本

- Minecraft: `1.20.1`
- Fabric Loader: `>=0.15.11`
- Fabric API: `>=0.92.0`

## 本模组分为个人版和服务器版

个人版移除了以下功能：

- 反作弊器
- 反作弊功能
- 在设置界面中的反作弊功能
- `Nbt`加密
- 检测


减少了模组的占用和不必要的内容

服务器版：

- 增加检测作弊后黑屏功能
- 增加检测出作弊十秒后踢出服务器功能

**本分支为个人版**

## 内容

### 钻石系列

四种新钻石：绿钻、黄钻、紫钻、粉钻。每种钻石都有对应的工具（剑、镐、斧、锹、锄）和盔甲。

- 绿钻：基础属性，适合前期
- 黄钻：中等属性，附带火焰附加
- 紫钻：高属性，高级附魔
- 粉钻：顶级属性，全套效果

### 彩虹宝石系列

通过彩虹宝石合成的顶级装备，属性远超钻石系列。

- 彩虹宝石工具：高耐久、高挖掘速度
- 彩虹宝石盔甲：提供抗性提升效果
- 彩虹宝石弓：高伤害、自带火焰

### 七彩神剑

本模组最强武器，需要达成契约才能发挥全部能力。

**契约达成方式**：手持七彩神剑快速连续右键（10秒内两次），即可达成契约。

**契约后能力**：
- 无限耐久
- 瞬间挖掘任何方块
- 免疫伤害（持剑时）
- 可配置的飞行能力
- 可配置的速度加成
- 可配置的最大生命值

**攻击模式**（通过设置界面配置）：
- 安全模式：普通攻击
- 范围攻击模式：清除范围内所有实体
- 无限制模式：直接删除目标实体
- 攻击模式：双倍伤害+爆炸

**特殊模式**（通过按键切换）：
- 毁灭模式：一击删除实体
- 特殊攻击模式：10倍伤害
- 冰冻模式：冻结目标
- 治疗模式：治疗周围友方单位
- 剑气模式：释放直线剑气破坏方块和伤害实体

### 彩虹维度

使用彩虹钥匙可以在主世界和彩虹海之间传送。

彩虹海是一个独特的维度，包含：
- 彩虹宝石矿石
- 彩虹守卫者等敌对生物
- 彩虹遗迹结构

### 灾厄系列

在特定生物群系自然生成的灾厄怪物团队：
- 灾厄亡灵：基础单位，产生粒子效果
- 灾厄鬼影：飞行单位
- 灾厄鬼兵：近战单位，可瞬移
- 灾厄之风：小首领，可冲刺和召唤小怪

击败灾厄之风可获得“灾厄之风”物品，用于召唤灾厄鬼兵。


## 指令


- `/mfswordmod give <物品ID> [数量]` - 给予指定物品

- `/mfswordmod rainbow mode <0-3>` - 设置七彩神剑攻击模式

- `/mfswordmod rainbow range <1-64>` - 设置范围攻击范围

- `/mfswordmod rainbow damage <数值>` - 设置基础伤害

- `/mfswordmod rainbow contract` - 强制达成契约

- `/mfswordmod rainbow settings true` - 打开设置界面



## 按键绑定（客户端）

- `V`键：切换毁灭模式
- `K`键：切换特殊攻击模式
- `I`键：清除所有模式
- `J`键：冰冻模式
- `N`键：治疗模式
- `M`键：剑气模式
