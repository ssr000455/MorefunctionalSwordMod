# 更多扩展模组 (MoreFunctionalSwordMod)

一个为 Minecraft 1.20.1 设计的 Fabric 模组，添加了多种钻石工具、彩虹宝石系列装备、七彩神剑以及反作弊系统。

## 版本

- Minecraft: 1.20.1
- Fabric Loader: >=0.15.11
- Fabric API: >=0.92.0

## 新增内容

### 钻石系列

四种新钻石：绿钻、黄钻、紫钻、粉钻。每种钻石都有对应的工具（剑、镐、斧、锹、锄）和盔甲。

- 绿钻：基础属性，适合前期
- 黄钻：中等属性，附带火焰附加
- 紫钻：高属性，高级附魔
- 粉钻：顶级属性，全套效果

### 彩虹宝石系列

通过彩虹宝石合成的顶级装备，属性远超钻石系列。

- 彩虹宝石工具：高耐久、高挖掘速度
- 彩虹宝石盔甲：提供抗性提升效果
- 彩虹宝石弓：高伤害、自带火焰

### 七彩神剑

本模组最强武器，需要达成契约才能发挥全部能力。

**契约达成方式**：手持七彩神剑快速连续右键（10秒内两次），即可达成契约。

**契约后能力**：
- 无限耐久
- 瞬间挖掘任何方块
- 免疫伤害（持剑时）
- 可配置的飞行能力
- 可配置的速度加成
- 可配置的最大生命值

**攻击模式**（通过设置界面配置）：
- 安全模式：普通攻击
- 范围攻击模式：清除范围内所有实体
- 无限制模式：直接删除目标实体
- 攻击模式：双倍伤害+爆炸

**特殊模式**（通过按键切换）：
- 毁灭模式：一击删除实体
- 特殊攻击模式：10倍伤害
- 冰冻模式：冻结目标
- 治疗模式：治疗周围友方单位
- 剑气模式：释放直线剑气破坏方块和伤害实体

### 彩虹维度

使用彩虹钥匙可以在主世界和彩虹海之间传送。

彩虹海是一个独特的维度，包含：
- 彩虹宝石矿石
- 彩虹守卫者等敌对生物
- 彩虹遗迹结构

### 反作弊系统

新增物品“反作弊器”，手持时自动检测玩家异常行为。

**检测项目**：
- 移动检测：飞行、加速、高速移动
- 战斗检测：攻击频率限制（CPS）
- 破坏检测：方块破坏速度限制
- 内存扫描：生命值、移动速度异常
- NBT签名验证：防止数据篡改
- 药水效果异常检测
- 背包物品堆叠检测

**使用方式**：
- 手持反作弊器自动生效
- OP可右键切换严格模式
- 累计违规自动踢出

### 灾厄系列

在特定生物群系自然生成的灾厄怪物团队：
- 灾厄亡灵：基础单位，产生粒子效果
- 灾厄鬼影：飞行单位
- 灾厄鬼兵：近战单位，可瞬移
- 灾厄之风：小首领，可冲刺和召唤小怪

击败灾厄之风可获得“灾厄之风”物品，用于召唤灾厄鬼兵。

## 如何开始

1. 采集绿钻、黄钻、紫钻、粉钻矿石
2. 合成对应钻石工具和盔甲
3. 使用四种钻石合成彩虹宝石
4. 使用彩虹宝石合成彩虹宝石装备
5. 使用彩虹宝石和钻石剑合成七彩神剑
6. 达成契约获得完全能力

## 指令


/mfswordmod give <物品ID> [数量] - 给予指定物品
/mfswordmod rainbow mode <0-3> - 设置七彩神剑攻击模式
/mfswordmod rainbow range <1-64> - 设置范围攻击范围
/mfswordmod rainbow damage <数值> - 设置基础伤害
/mfswordmod rainbow contract - 强制达成契约
/mfswordmod rainbow settings true - 打开设置界面


## 按键绑定（客户端）

- V键：切换毁灭模式
- K键：切换特殊攻击模式
- I键：清除所有模式
- J键：冰冻模式
- N键：治疗模式
- M键：剑气模式
